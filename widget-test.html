🤖 Salvebot Widget Demo

This page demonstrates the Salvebot chatbot widget in action!

✨ Features Included

  • 🎨 Beautiful, modern UI with animations
  • 📱 Fully mobile responsive design
  • 💬 Real-time chat with AI assistant
  • 📚 RAG-powered document knowledge retrieval
  • ⌨️ Typing indicators and smooth animations
  • 🕐 Message timestamps
  • ✅ Confidence indicators for AI responses
  • 🎯 Domain verification and security

🚀 How to Test

Look for the floating chat button in the bottom-right corner of this page!

Click it to start chatting with the AI assistant.

Note: You'll need a valid chatbot ID and verified domain for the widget to work with real data.

📋 Integration Code

To add this widget to your website, simply include this script tag:

<script src="https://salvebot-api.fideleamazing.workers.dev/api/chat/widget.js" data-chatbot-id="YOUR_CHATBOT_ID" data-domain="yourdomain.com"></script>
